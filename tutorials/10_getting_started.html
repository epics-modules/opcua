Getting Started

This tutorial guides you through the process of building the OPC UA Device Support and setting up a basic EPICS IOC connection.

Prerequisites

  • A C++ Compiler that supports the C++11 standard. Microsoft Visual C++ needs to be from Visual Studio 2015 or newer. g++ needs to be 4.6 or above.

  • [EPICS Base][epics-base]. EPICS Base 3.15 (>= 3.15.7) or EPICS 7 (>= 7.0.4).

  • A low-level OPC UA client library. Two options are supported: Either the free open-source client of the [open62541 project SDK][open62541] or the commercially available [Unified Automation C++ Based OPC UA Client SDK][uasdk].

  • (Optional:) The [EPICS gtest module][epics-gtest]. Needed if you want to build and run the unit tests.

To use the OPC UA Device Support, you first need to get and/or build the prerequisites, then the Device Support module itself, then your IOC application that uses it.

Linux and Windows native builds are supported.
